基础组织架构

一、概览

  • 集团-Client
  • 公司代码-Company Code
  • 销售组织-Sale Organization

二、集团

  1. 最大的组织级别，包含多个公司代码

三、公司代码

  1. 一个法人实体，是财务会计的最基本单元。
  2. 最小的外部会计核算组织单位
  3. 法律实体和独立的会计核算单位
  4. 可以创建三大表：资产负债表、损益表

四、销售组织

  1. 销售与分销的最基本单元
  2. 一个公司代码下可以包含多个销售组织
